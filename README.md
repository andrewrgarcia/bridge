# bridge

Connecting array data between Python and C++ through the use of JSON files. DOK = Dictionary of Keys (json)

[MIT License - Copyright (c) 2022 Andrew Garcia, PhD](LICENSE)

```ruby
array to DOK:
{
	"map": [
		[1],
		[4],
		[5],
		[8],
		[9]
	],
	"value": [2, 3, 2, 2, 2],
	"odims": [10]
}

DOK to array:
[0. 2. 0. 0. 3. 2. 0. 0. 2. 2.]
```
